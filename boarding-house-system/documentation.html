Boarding House Management System Documentation

System Overview

A complete web-based management system for boarding houses with:

  • Tenant management
  • Room management
  • Payment tracking
  • Maintenance requests
  • Dashboard analytics

Key Features

1. Tenant Management

  • Add/edit/delete tenant records
  • View tenant details
  • Search and filter functionality
  • Responsive design for all devices

2. Room Management

  • Room status tracking (available/occupied/maintenance)
  • Capacity management
  • Floor-wise organization
  • Visual indicators for room status

Technical Implementation

Frontend

  • HTML5, CSS3, JavaScript
  • Tailwind CSS for styling
  • Chart.js for visualizations
  • Font Awesome icons

Backend API Endpoints

Database Connection Setup

1. MongoDB Installation

Install MongoDB Community Edition:

  • Windows: Download installer from mongodb.com
  • Mac: brew install mongodb-community
  • Linux: sudo apt-get install mongodb

2. Database Configuration

Edit the backend configuration file:

// In server.js
const mongoose = require('mongoose');
mongoose.connect('mongodb://localhost:27017/boardingHouseDB', {
  useNewUrlParser: true,
  useUnifiedTopology: true
});

3. Environment Variables

Create a .env file in your backend directory:

DB_URI=mongodb://localhost:27017/boardingHouseDB
DB_USER=your_username
DB_PASS=your_password
JWT_SECRET=your_jwt_secret

4. Database Models

Example User model (models/User.js):

const mongoose = require('mongoose');
const userSchema = new mongoose.Schema({
  name: String,
  email: { type: String, unique: true },
  password: String,
  role: String
});
module.exports = mongoose.model('User', userSchema);

Getting Started

  1. Install dependencies: npm install
  2. Start MongoDB service: mongod
  3. Start backend server: node server.js
  4. Open frontend: python3 -m http.server 8000 -d frontend
  5. Access system at: http://localhost:8000
